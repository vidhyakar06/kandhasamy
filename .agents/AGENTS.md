# Kandhasamy Textile Rules

## Environment Guidelines
- The application is hosted on Vercel at https://kandhasamy.vercel.app and the codebase is pushed to GitHub at https://github.com/vidhyakar06/kandhasamy.
- Do not run local servers or suggest localhost URLs to the user.
- All testing, verification, and browser automation actions should be performed against the live Vercel site: https://kandhasamy.vercel.app.
